# 🚀 100 Days of Code - 


## 📌 About the Challenge
This repository is part of the **100 Days of Code 
Data Structures & Algorithms (DSA)** challenge assigned by my college.

In this challenge:
- 📅 **Duration:** 100 Days  
- 🧩 **Daily Task:** Solve **2 DSA problems every day**  
- 💻 **Language:** C / C++ / Python  
- 🎯 **Goal:** Improve problem-solving skills and build strong DSA fundamentals
